2d transform

misol
transform: rotate(20deg); transform: rotate(-20deg);

misol

top-left;
right-buttom;
(30px, 40px);


translate

translate elementni hozirgi joyidan surish uchun ishlatiladi
misol
div{
transform: translate(x, y)
}


scale

transform scale elementni o'lchovini o'zgartirish uchun ishlatiladi
(0;1) qiymat elementt o'lchamini kamaytiradi
1 dan katta qiymatda element o'lchami kattalashadi


skew

skew elemntni ma'lum bir darajaga (deg) og'dirish uchun ishlatiladi
mandiy qiymat og'ish burchagini teskari tomonga o'zgartiradi
misol:
transform: skew(x, y);
skew (20deg, 30deg);


perspective

element userdan qanday uzoqlikda turganini belgilab beradi qiymat qancha kichik bo'lsa element foydalanuvchiga shuncha yaqin bo'ladi
misol
div{
transform: perspective(40px) translate3d(0px, -30px, -20px);
}


transition

misol

transition: all 1.3s;
